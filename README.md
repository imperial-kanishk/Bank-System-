# Banking System (Java)

## Overview

This is a console-based Banking System developed in Java using Object-Oriented Programming principles. The application allows users to create savings and current accounts, deposit and withdraw funds, transfer money between accounts, and view transaction history.

## Features

* Create Savings Accounts
* Create Current Accounts
* Deposit Money
* Withdraw Money
* Transfer Funds Between Accounts
* Check Account Balance
* View Transaction History
* Generate Account Statements

## Technologies Used

* Java
* OOP (Inheritance, Abstraction, Polymorphism, Encapsulation)
* ArrayList
* HashMap

## Project Structure

entities/

* BankAccount
* SavingsAccount
* CurrentAccount
* Transaction

service/

* Bank

Main.java

## How to Run

1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Compile and run Main.java.
4. Use the menu-driven interface to perform banking operations.
